◈ URGENCY ADAPTATION
NeurIPS 2024 Concordia Contest: LLM agents handle vanilla negotiation but fail when scenarios require detecting and adapting to peers with different urgency levels. Heterogeneous priority is invisible to them.

◈ SIMULTANEOUS COMMITMENT
4 agents commit simultaneously. No agent can wait to observe others before acting. The symmetry is structural — identical reasoning leads to identical actions, which collide. Training must break this symmetry.

◈ 03 / REWARD DESIGN
Reward Design
◈ FOUR INDEPENDENT COMPONENTS — GAMING ONE DOESN'T EARN THE OTHERS
R1 · 0.5
SERVICE RESTORED
Primary signal. Agent's service fixed within deadline. Must exceed 0.5 threshold for binary reward. Cannot be gamed by MONITOR — no deployment = no credit.
R2 · 0.3
SITE UPTIME
Group welfare. Fraction of team services restored across the episode. Prevents selfish strategies — ignoring peers costs R2 even when R1 is perfect.
R3 · 0.1
CLEAN DEPLOYS
Collision avoidance. Fraction of deploys that were collision-free. Never deployed = 0.0 (not 0.5) — idle agents get no credit for avoiding collisions they never risked.
R4 · 0.1
YIELD TO CRITICAL
Urgency adaptation. Did agent yield (MONITOR) when a peer had a tighter deadline margin and was about to contend for the same server? Hardest signal — shows social reasoning.
◈ BINARY GRPO REWARD FORMULA
reward = 1.0 if (r1 > 0.5 and 0.5·r1 + 0.3·r2 + 0.1·r3 + 0.1·r4 > 0.35) else 0.0

◈ 05 / TRAINING CONFIGURATION
Training Configuration
◈ QWEN-2.5-3B-INSTRUCT · GRPO · UNSLOTH · A100 40GB

Motivation: DPBench (Feb 2026) showed frontier LLMs deadlock above 95% on simultaneous coordination tasks — adding a communication channel made it worse, not better. NeurIPS 2024 Concordia Contest showed LLM agents fail when peers have heterogeneous urgency levels. TICKETMELT targets both failure modes in one training scenario.

Training signal: Binary GRPO reward — 1.0 when service restored within deadline and composite score above threshold, 0.0 otherwise. Continuous component breakdown logged for analysis. No expert demonstrations. No supervised warmup. Pure RL from environment interaction.
